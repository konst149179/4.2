Проект - Windows-приложение на C# (.NET Framework / .NET) для управления тарифами на железнодорожные направления с поддержкой скидок, сортировки, поиска и сохранения данных.

Структура проекта
DataClasses.cs
Содержит основные классы бизнес-логики:

DiscountStrategy — абстрактный класс стратегии расчёта скидки.

NoDiscount, PercentageDiscount — реализации стратегий (без скидки и процентная скидка).

Tariff — информация о направлении, базовой и итоговой стоимости, применённой стратегии скидки.

Station — хранилище тарифов с методами добавления, удаления, изменения и поиска направлений с минимальной стоимостью.

MainForm.cs и MainForm.Designer.cs
Главное окно приложения:

Отображает список тарифов в DataGridView.

Поддерживает сортировку по всем колонкам (включая скидки).

Содержит кнопки:

Добавить / Изменить / Удалить тариф, Найти направления с минимальной стоимостью, Сохранить / Загрузить данные из файла.

EditForm.cs и EditForm.Designer.cs
Форма для добавления нового или редактирования существующего тарифа:

Ввод направления и базовой стоимости.

Возможность указать процент скидки (опционально).

FileService.cs
Служебный класс для сериализации и десериализации данных:

Сохраняет тарифы в текстовый файл в формате направление|стоимость|тип_скидки|процент.

Загружает данные из файла, воссоздавая объекты Tariff.

Program.cs
Точка входа в приложение. Запускает главное окно.
